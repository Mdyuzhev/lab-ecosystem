# /push — Быстрый пуш

Добавляет все изменения, коммитит и пушит.

## Использование

```
/push [message]
```

## Шаги

```bash
git add -A
git status --short
git commit -m "$MESSAGE"
git push origin main
```

## Примеры

```
/push fix responsive layout
/push add biotech lab details
/push update economics data
```
