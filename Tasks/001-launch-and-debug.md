# Задача: Запуск и отладка приложения Lab Ecosystem

## Контекст

Проект представляет собой React-приложение, собираемое с помощью Vite. Открытие `index.html` напрямую в браузере не работает, потому что:
1. Vite использует ES-модули, которые требуют HTTP-сервера
2. JSX-код не компилируется без сборщика
3. Tailwind CSS не обрабатывается без PostCSS pipeline

## Шаги для запуска

### Шаг 1: Перейти в директорию проекта

```bash
cd E:\Politech\lab-ecosystem
```

### Шаг 2: Установить зависимости

```bash
npm install
```

Это установит React, Vite, Tailwind и все необходимые пакеты из `package.json`.

### Шаг 3: Запустить dev-сервер

```bash
npm run dev
```

После запуска в консоли появится:
```
  VITE v5.0.8  ready in XXX ms

  ➜  Local:   http://localhost:3000/
  ➜  Network: http://192.168.x.x:3000/
```

### Шаг 4: Открыть в браузере

Перейти по адресу `http://localhost:3000/` — сайт откроется автоматически (настроено в vite.config.js).

## Возможные ошибки и решения

### Ошибка: "npm не является внутренней или внешней командой"

Node.js не установлен или не добавлен в PATH. Решение:
1. Скачать Node.js с https://nodejs.org/ (LTS версия)
2. Установить с галочкой "Add to PATH"
3. Перезапустить терминал

### Ошибка: "Cannot find module 'react'"

Зависимости не установлены. Решение:
```bash
npm install
```

### Ошибка: "Port 3000 is already in use"

Порт занят другим приложением. Решения:
1. Закрыть приложение, использующее порт 3000
2. Или изменить порт в `vite.config.js`:
```javascript
server: {
  port: 3001,  // другой порт
  open: true
}
```

### Белый экран без ошибок

Открыть DevTools браузера (F12) → вкладка Console. Там будут ошибки JavaScript, если что-то сломалось в компоненте.

При успешном запуске должна отобразиться интерактивная схема с:
- Заголовком "Экосистема R&D лабораторий"

- Управляющим комитетом посередине
- Тремя карточками лабораторий (ИИ, ПО, Робототехника)
- Двумя будущими лабораториями справа (Биотех, Химия)

## Hot Reload

Vite поддерживает горячую перезагрузку. При изменении файлов в `src/` браузер автоматически обновится без перезапуска сервера.
